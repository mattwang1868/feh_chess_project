# feh_chess_project
Fully playable chess game created in C++ as part of the Fundamentals of Engineering for Honors course sequence.

## Contributors
- Matt Wang
- Owen Goodwin

## Features
- Fully playable chess game created using OSU's FEH libraries
- Checks for and only allows for legal moves
- Detection of check and checkmate
- Player time limits
- Computer opponent that plays random moves
- Same-device 2-player game
